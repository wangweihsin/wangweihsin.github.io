本研究以數位智能法院的發展為研究對象，分別針對民事法院、家事法院、商業法院、憲法法院以及刑事法院五大法院系統。

本案應用實證研究與法律科技方法，研究法院程序之數位化及法院裁判形成。具體議題包括（ㄧ）民事法院的線上訴訟與數位證據調查、程序公平性之裁判基準以及證據保全與定暫時狀態處分之利益衡量判斷；（二）家事法院的未成年子女扶養費裁判、老親扶養費裁判以及扶養費算定；（三）商業法院的公司經營權定暫時狀態假處分、股份收買公平價格裁定幾集電子書狀；（四）憲法法院的司法積極主義、大法官解釋標的認定以及憲法法庭裁判權的認定；以及（五）刑事法院的數位證據、遠距審判以及司法系統決策。本研究將基於對上述五大法院系統的研究，形成整合性的觀點。

數位智能法院、
法律科技與接近正義

Intelligent

檢視司法院的數位智能法院政策，分析法院程序電子化、數位化、線上化對於訴訟行為之影響，建構法理基礎，作為長遠政策發展的參考依據，並與國際接軌。

Empirical

彙整實證資料並分析裁判模式，此一方面可作為預測未來裁判的基礎，一方面也可做為檢視裁判妥適性的依據；此外本研究成果累積的實證資料亦可作為進一步發展法律科技工具的原料。

Interdisplinary

展開法律資料分析學，藉由法律學者與資訊科學學者兼的跨領域合作，將資料分析技術應用於法律制度研究，與國際法律資料學研究對話。 

Human Right

一方面探究司法程序之數位化、智能化及法律科技對於訴訟權保障、正當程序、權利保護及接近正義之影響，一方面結合法律科技之應用，就特定類型知識漸進一部歸納裁判模式及研究可行的裁判預測系統。
